# debug-agent

> Paste a stack trace, get a root cause analysis and a fix.

## What it does

Reads your stack trace, fetches relevant source lines, searches docs and known issues, and explains exactly what went wrong and how to fix it. Works with Python, JS, Java, Go.

## Quickstart

```bash
pip install debug-agent-ai
```

## Usage

```bash
debug-agent analyze --trace error.log --src src/
```
